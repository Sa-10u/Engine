18 佐藤莉雄樹(220350)

先生方からご指示いただいたものは実装済み。

キーボードではなく、マウスで操作可能とし、shiftをおすと微調整できるようにした。テンキーの1を押すと初期の場所に戻るので安心して操作の練習ができる。いっぺんに地形を上げたいときようにショートカットキーも搭載。一番工夫したのは、地形の上げ下げはクリックであるがブロックの変更は長押しでも可能であるところ。

初期の場所に戻すことはできたが、残念ながらターゲット(カメラの向き)を変えることはなぜかできなかった。10Ｆかけてゆっくり戻したいがターゲットがエラーを吐く。そのためポジションしか10Fかけてもどってくれない。バグの箇所や式の作り方が非常に楽しかった。
